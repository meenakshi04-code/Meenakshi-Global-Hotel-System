# 🌍 Meenakshi Global Hotel System

A Java-based Hotel Management System with global room booking features.

## ✨ Features

* View available rooms
* Search rooms by country (India, Dubai, Korea)
* Book rooms
* View bookings
* Simple billing system

## 💻 Technologies Used

* Java
* OOP Concepts

## 🚀 How to Run

1. Compile:
   javac Main.java

2. Run:
   java Main
